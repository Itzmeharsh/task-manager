# 🚀 Mini SaaS Task Management System

## 🔗 Live Demo
https://task-manager-vert-psi.vercel.app/

---

## 📌 Features

### 🔐 Authentication
- User Signup & Login
- JWT-based authentication
- Password hashing (bcryptjs)
- Protected routes

### 📋 Task Management
- Create tasks
- View personal tasks only
- Mark tasks as completed
- Delete tasks

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- Node.js
- Express.js
- Sequelize ORM

### Database
- PostgreSQL (Supabase)

### Deployment
- Frontend: Vercel
- Backend: Render

---

## 📂 Project Structure

task-manager/
├── task-manager-backend/
├── client/

---

## ⚙️ Environment Variables

Backend (.env)

DB_URL=your_database_url  
JWT_SECRET=your_secret  

---

## 🚀 How to Run Locally

### Backend
cd task-manager-backend  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm run dev
